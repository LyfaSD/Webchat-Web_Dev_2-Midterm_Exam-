# MiniSnap Frontend

A basic Snapchat-like app built with React + Vite.

## Features
- Register / Login with JWT auth
- Post stories with images and captions
- View all stories
- Chat with other users

## Setup

1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your backend URL
4. Run `npm run dev`

## Deployment
Deployed on Vercel. Set `VITE_API_URL` in Vercel's environment variables.
